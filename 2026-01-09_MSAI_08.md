## What I Learned

### Apollo Mission Context data analysis & Pandas Operations
- `df.rename(columns={...})` returns new df — need reassignment or `inplace=True`
- `groupby().mean(numeric_only=True)` required in newer pandas
- Merge on index: `left_index=True, right_index=True` or `.join()`

## Problems I Hit
| Problem | Solution |
|---------|----------|
| `'data\rocksamples.csv'` → OSError | Use `/` instead of `\` (escape character issue) |
| Git push rejected | `git pull origin main` first, then push |
| `rename()` not changing df | Add `inplace=True` or reassign result |
| Pie chart `autopct='%.f%%'` error | Use `'%.0f%%'` or `'%d%%'` |
| f-string quote collision `f'{df['col']}'` | Use different quotes: `f"{df['col']}"` |

**Validate data flow** — ensure all metrics come from consistent source
