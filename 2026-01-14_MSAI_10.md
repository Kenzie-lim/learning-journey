## What I Learned

### 1. Data Loading & Concatenation
- Load multiple CSV files with `pd.read_csv()` using `encoding='cp949'` for Korean text
- Use `fr'path\{variable}.csv'` for raw + f-string combination (order doesn't matter: `fr` or `rf`)
- Concatenate DataFrames with `pd.concat([df1, df2, ...])`

### 2. Data Preprocessing
- Parse dates
- Extract datetime components: `.dt.day`, `.dt.hour`, `.dt.dayofweek`
- Map values

### 3. Seaborn Visualization
- `countplot`: Count rows per category (single axis only, x OR y)
- `barplot`: Aggregate values (e.g., mean) across categories
- `heatmap`: Best for pivot table visualization
- Color customization: `palette='Set2'` or dict for specific colors

### 4. Matplotlib Korean Font Setup
```python
plt.rc('font', family='Malgun Gothic', size=15)  # Windows default Korean font
```

### 5. Pivot Table
- `index`: row labels, `columns`: column labels, `values`: data to aggregate
- `aggfunc='size'` for counting, `aggfunc='mean'` for averages
- Cannot aggregate string columns â†’ specify numeric `values` column

### 6. Folium Map Visualization
- Basic map: `folium.Map(location=[lat, lon], zoom_start=10)`
- Markers: `folium.Marker([lat, lon], icon=folium.Icon(color='red'))`
- CircleMarker: `folium.CircleMarker([lat, lon], radius=5, color='blue')`
- Choropleth: Requires GeoJSON + data matching via `key_on`
